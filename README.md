Nice Gadjets - a store selling electronic equipment such as phones, tablets, and electronic watches.

This project was created as a portfolio application, focusing on clean architecture, responsive UI, and smooth user experience. The goal was to build a visually appealing and user-friendly product catalog similar to a real-world e-commerce website.

Live demo:
    - [Demo link](https://vitalii120296.github.io/phone_catalog_store/)

Design Reference:
    - [Link](https://www.figma.com/design/BUusqCIMAWALqfBahnyIiH/Phone-catalog--V2--Original-Dark?node-id=0-1&p=f)


Technologies used:

- React
- React DOM
- HTML5
- SCSS (modular structure)
- JavaScript (ES6+)
- BEM methodology
- Git & GitHub Pages

Features:

- Catalog of electronic products (phones, tablets, watches)
- Responsive design (mobile / tablet / desktop)
- Clean and modern UI
- Component-based architecture (React)
- User-friendly navigation and page structure
- Optimized layout and semantic HTML
- No UI frameworks used


Getting Started:

1. Clone the repository:
git clone https://github.com/Vitalii120296/phone_catalog_store.git
cd 2048_game_js

2. Install dependencies:
npm install
# or
yarn install

3. Run the project locally:
npm start
# or
yarn start
